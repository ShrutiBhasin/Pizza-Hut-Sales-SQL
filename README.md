# 🍕 Pizza Hut Sales Analysis - SQL Project

## About
Analyzed Pizza Hut sales data using MySQL Workbench to uncover key business insights.

## Dataset
- 21,350+ orders
- 4 Tables: orders, orders_details, pizzas, pizza_types

## Questions Solved
- Total orders placed
- Total revenue generated
- Highest priced pizza
- Most common pizza size
- Top 5 most ordered pizzas
- Category wise quantity ordered
- Orders by hour
- Average pizzas ordered per day
- Top 3 pizzas by revenue per category
- Percentage revenue contribution by category
- Cumulative revenue over time

## Tools Used
- MySQL Workbench

## Skills
Joins | Aggregations | Subqueries | Window Functions | Group By
